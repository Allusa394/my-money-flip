## 1. СУТЬ ПРОЕКТА И ЦЕЛЬ

### Что это
**The Money Flip** — интерактивный трансформационный сервис, который помогает людям заменять ограничивающие убеждения о деньгах на ресурсные, используя НЛП-рефрейминг через ИИ (Claude API).

Одновременно — **портфолио-страница автора**: цифрового мастера, который создаёт сайты, приложения и ИИ-агентов. Сам сервис — главный кейс портфолио, живое доказательство экспертизы.

### Двойная цель
| Цель 1 | Цель 2 |
|--------|--------|
| Конвертировать пользователя в подписчика/клиента инструмента | Конвертировать потенциального клиента в заявку на разработку |

### Уникальная позиция
Специализированных AI-инструментов для НЛП-трансформации денежных убеждений **на русском языке не существует**. The Money Flip занимает эту нишу первым.

## 2. ЦЕЛЕВАЯ АУДИТОРИЯ

### Первичная аудитория (инструмент трансформации)
- **Пол/возраст:** Женщины 28–45 лет (65–70% сегмента), мужчины 30–50 лет
- **Психографика:** Работают над собой, читают книги по психологии денег, сталкивались с НЛП или коучингом, не хотят «инфобизнес», но верят в работу с подсознанием
- **Боль:** Зарабатывают меньше, чем хотят. Понимают, что «что-то мешает изнутри». Устали от аффирмаций без результата
- **Триггер входа:** «Хочу разобрать конкретное убеждение прямо сейчас»
- **Антитриггер:** «Очередной инфобизнес», «манифестации», «притяжение денег»

### Вторичная аудитория (клиенты на разработку)
- **Кто:** Предприниматели, маркетологи, онлайн-эксперты, которым нужны сайт / приложение / ИИ-агент
- **Боль:** Нужен подрядчик, которому можно доверять. Хотят видеть, что человек делает
- **Триггер доверия:** Портфолио = живое рабочее приложение + кейсы с метриками

> **Замените ограничивающее убеждение о деньгах за 3 минуты**

**Избегать:**
- «богатство» / «успех» в заголовках — маркер дешёвого инфобизнеса
- «манифестация», «закон притяжения», «миллион» в заголовках

#### Карточка инструмента (форма):

**Поле 01 — Убеждение:**
- Label: `01 · Какая мысль о деньгах возвращается снова и снова?`
- Placeholder: `Например: «Деньги достаются только тяжёлым трудом» или «Я не заслуживаю больших денег»`
- Тип: `<textarea>`, min-height 110px
- Поведение: автозаполнение при клике на тег из секции 04

**Поле 02 — Контекст:**
- Label: `02 · В какой сфере это мешает больше всего?`
- Тип: `<select>`
- Опции:
  ```
  — Выберите контекст —
  💼 Карьера / Работа по найму
  🚀 Бизнес / Предпринимательство
  🏠 Семья / Отношения с деньгами дома
  🎯 Личные финансы / Инвестиции
  ```

**Поле 03 — Глубина убеждения:**
- Label: `03 · Насколько сильно это убеждение влияет на вашу жизнь?`
- Тип: кастомный слайдер 1–10
- НЕ «0-10» и «слабое/корневое» — заменить на 3 якоря:
  ```
  1–3: «Замечаю, но не сильно мешает»
  4–6: «Регулярно ограничивает мои действия»
  7–10: «Живу с этим всю жизнь — кажется, что изменить нельзя»
  ```
- Визуальный якорь под слайдером: подсказка меняется в зависимости от позиции

**Кнопка:**
- Текст: `⚡ Переписать это убеждение`
- Цвет: энерго-акцент (#B8FF00)
- Состояния: default / loading (dots) / success

**Валидация (обязательно):**
```
- Пустое поле → красный border + «Опишите убеждение, чтобы начать трансформацию»
- Контекст не выбран → «Выберите контекст — это влияет на стратегию рефрейминга»
```

#### Блоки результата (3 блока, появляются одновременно после ответа API)

Каждый блок — отдельный AI-запрос с разным фокусом языка по методологии Пелехатого и Спирицы.
Все три блока запускаются параллельно (`Promise.all`) и отображаются с stagger-анимацией.

### Структура API-вызова (Serverless Function — 3 параллельных запроса)

```javascript
// /api/transform.js (Vercel Edge Function)
export default async function handler(req, res) {
  const { belief, context, intensity } = req.body;

  // Rate limiting + validation...

  const baseSystem = buildBaseSystem(context, intensity);

  // Три параллельных вызова — по одному на каждый блок
  const [master, battle, code] = await Promise.all([
    anthropic.messages.create({
      model: 'claude-haiku-4-5-20251001',
      max_tokens: 300,
      system: baseSystem + PROMPT_MASTER,  // Ракурс Мастера
      messages: [{ role: 'user', content: belief }]
    }),
    anthropic.messages.create({
      model: 'claude-haiku-4-5-20251001',
      max_tokens: 250,
      system: baseSystem + PROMPT_BATTLE,  // Тест на прочность
      messages: [{ role: 'user', content: belief }]
    }),
    anthropic.messages.create({
      model: 'claude-haiku-4-5-20251001',
      max_tokens: 300,
      system: baseSystem + PROMPT_CODE,    // Код Новой Реальности
      messages: [{ role: 'user', content: belief }]
    })
  ]);

  res.json({
    master: master.content[0].text,
    battle: battle.content[0].text,
    code:   code.content[0].text
  });
}
```
