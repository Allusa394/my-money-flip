## Что это за проект

**The Money Flip** — интерактивный трансформационный сервис для работы с ограничивающими убеждениями о деньгах через НЛП + Claude AI.

Двойная цель:
1. Инструмент трансформации убеждений для конечных пользователей
2. Портфолио-страница автора (цифровой мастер: сайты, приложения, ИИ-агенты)

**Ниша свободна**: специализированных AI-инструментов для НЛП-трансформации денежных убеждений на русском языке не существует.

## AI-интеграция (когда будем подключать)

**Модель:** `claude-haiku-4-5-20251001` — быстро и дёшево (~$0.0008 за 3 параллельных запроса)

**Архитектура:** Serverless function (Vercel Edge / Netlify) — API-ключ никогда в клиентском коде.

**Три параллельных запроса** через `Promise.all` — по одному на каждый блок с разным системным промптом.

Готовые системные промпты — в `brief.md`, раздел 8 «AI-интеграция».

## Правила работы с кодом

- Проект работает **без сборки** — открывается двойным кликом на `index.html`
- Tailwind CDN + Google Fonts CDN (нужен интернет)
- Не добавлять npm, webpack, vite — только ванильный стек
- Все CSS custom properties в `:root` блоке `styles.css`
- Mock данные в `script.js` — объекты `mockMaster`, `mockBattle`, `mockCode`
- При замене mock на реальный API — создать `/api/transform.js` отдельно

## Целевая аудитория

**Первичная (инструмент):** Женщины 28–45 лет (65–70%), работают над собой, знакомы с НЛП/коучингом, устали от аффирмаций без результата.

**Вторичная (портфолио):** Предприниматели и маркетологи, которым нужны сайт / приложение / ИИ-агент.
